# Floe

Mobile-first prototype for a Duolingo-like career-switching trainer, starting with quant finance interview prep.

## Prototype Loop

- Quant Finance Interview track
- Daily mode: 3 quick drills and 1 deeper puzzle
- Endless mode for extra practice
- Wrong-answer misconception debriefs
- Playful cuttlefish characters for mentor, skeptic, drill, and explorer roles
- Local streak/readiness state
- Basic PWA manifest and service worker

## Run

```bash
npm install
npm run dev -- --host 0.0.0.0
```

Then open `http://localhost:5173/`.

## Build

```bash
npm run build
```
