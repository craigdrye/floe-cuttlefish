# City Skylines Asset Plan

This folder holds PNG-backed `Guess the City Skyline` assets. The existing course already has twelve skyline questions; this plan replaces SVG clue cards with cropped PNG skyline images in batches.

## Generation Rules

- Use 2x2 source grids, then crop into four square PNGs.
- Use recognizable city silhouettes and landmark combinations, not text labels.
- Prefer clean travel-poster / geography-field-guide compositions with high visual contrast.
- Keep each city visually distinct by combining skyline shape, landmark, water/mountain/desert cues, and architectural profile.
- Avoid readable signs, captions, UI, flags, logos, watermarks, and modern brand marks.
- Keep filenames lowercase kebab-case.
