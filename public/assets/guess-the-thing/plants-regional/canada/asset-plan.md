# Canada Plants Asset Plan

Target: first 15 PNG-backed quiz assets for `plants-regional/canada`.

## Naming Rule

- Use lowercase kebab-case.
- Use stable common names.
- Default file pattern: `common-name.png`
- If variants are needed later:
  - `common-name-alt-1.png`
  - `common-name-flowering.png`
  - `common-name-winter.png`
