# Class 3 Math
**ID**: `col-class-3-math` · **Discipline**: Maths · **Year**: 3 (ages 7–8)
**Frameworks**: UK KS2 Year 3, US Common Core Grade 3, ACARA v9 Year 3, India NCERT Class 3

## Course Philosophy
Year 3 is the "gear change" year. Children transition from KS1 concrete methods to formal written strategies, and multiplication/division become central. This is where number fluency either cements or cracks — every module drives toward confident, flexible calculation.

## Module 1: Place Value & Number to 1000 (20%)
- Read, write, order, and compare numbers to 1000
- Place value: hundreds, tens, ones — partition and recombine (e.g. 372 = 300 + 70 + 2)
- Count in multiples of 4, 8, 50, and 100 from any starting number
- Round to the nearest 10 and 100
- Estimate answers before calculating
- Negative numbers in context (temperature, lifts, number lines)
- **Questions**: Partitioning, ordering, rounding, "what is 10 more/less than 476?"

## Module 2: Addition & Subtraction (20%)
- Formal columnar addition and subtraction (with regrouping/exchanging)
- Add and subtract mentally: 3-digit ± 1s, 10s, 100s
- Inverse operations to check answers
- Solve two-step word problems deciding which operations to use
- Estimate to check: "Is 287 + 345 closer to 500 or 700?"
- **Questions**: Column method, word problems, inverse checking, estimation

## Module 3: Multiplication & Division (25%)
- Recall all multiplication and division facts for 3×, 4×, 8× tables
- Consolidate 2×, 5×, 10× (instant recall)
- Multiply 2-digit by 1-digit using partitioning (23 × 4 = 20×4 + 3×4)
- Division with remainders (e.g. 14 ÷ 3 = 4 r 2)
- Understand multiplication as scaling: "3 times as many"
- Properties: commutativity (a×b = b×a), associativity
- **Questions**: Table recall, grid multiplication, division with remainders, word problems

## Module 4: Fractions (15%)
- Count up and down in tenths
- Recognise, find, and write fractions: ½, ⅓, ¼, ⅕, ⅙, ⅛, 1/10
- Recognise and show equivalent fractions (½ = 2/4 = 3/6)
- Add and subtract fractions with the same denominator within one whole
- Compare and order unit fractions (smaller denominator = bigger fraction)
- Fractions on a number line
- **Questions**: Equivalent fraction matching, fraction addition, ordering, number line placement

## Module 5: Measurement, Geometry & Statistics (20%)
- Perimeter: measure and calculate perimeter of simple 2D shapes
- Measure and compare lengths (mm, cm, m), mass (g, kg), capacity (ml, l)
- Tell time to nearest minute; use 12-hour and 24-hour clocks; calculate durations
- Roman numerals I to XII
- Identify right angles; recognise perpendicular and parallel lines
- Horizontal, vertical, and oblique lines
- Interpret and present data using bar charts, pictograms, and tables
- **Questions**: Perimeter calculation, unit conversion, clock reading, right angle identification, bar chart interpretation

## Stretch Zone
- 6× and 7× tables introduction
- Multiply 2-digit by 1-digit using short multiplication (formal)
- Equivalent fractions with different denominators (½ = 4/8)
- Introduction to the 24-hour clock for timetables
- Area as "counting squares" for rectilinear shapes
- Interpret data to solve two-step problems
