## Course Philosophy
This syllabus integrates the broad, conceptual framework of AP Computer Science Principles with the rigorous, object-oriented programming depth of AP Computer Science A (which uses Java).

## Module 1: CS Principles - The Internet, Data, and Impact (15%)
- The Internet: Protocols (TCP/IP, UDP, HTTP), routing, fault tolerance, DNS, and the physical network
- Cybersecurity: Symmetric/Public Key encryption, malware, phishing, DDoS attacks
- Digital Information: Binary representation, hexadecimal, ASCII, abstraction, lossy vs lossless compression
- Global Impact: Bias in algorithms, the digital divide, crowdsourcing, legal/ethical concerns (copyright, open source)
- **Questions**: Explaining how public key cryptography works, converting binary to hex, analyzing algorithmic bias

## Module 2: CS A - Programming Fundamentals (Java) (20%)
- Primitive types (`int`, `double`, `boolean`) vs Reference types
- Using Objects: creating objects with `new`, calling methods, the `String` class (methods like `substring`, `indexOf`, `equals`), the `Math` class
- Boolean Expressions and `if/else` Statements (short-circuit evaluation, De Morgan's Laws)
- Iteration: `while` loops, `for` loops, enhanced `for` loops (for-each)
- **Questions**: Tracing complex nested loops, evaluating boolean expressions using De Morgan's laws, using `String` methods correctly

## Module 3: CS A - Object-Oriented Programming (25%)
- Writing Classes: Instance variables (attributes), constructors, methods (behaviors)
- Encapsulation: `private` vs `public` access modifiers, getter/setter methods
- The `this` keyword and static variables/methods
- Inheritance: Subclasses and superclasses, the `extends` keyword, overriding methods, using `super()`
- Polymorphism: Declared type vs actual type, dynamic method binding
- **Questions**: Designing a class hierarchy, predicting the output of polymorphic method calls, tracing constructor execution

## Module 4: CS A - Data Structures (Arrays and ArrayLists) (20%)
- 1D Arrays: Creation, initialization, array traversal, accessing elements, `ArrayIndexOutOfBoundsException`
- The `ArrayList` class: Generic types (`ArrayList<E>`), auto-boxing/unboxing (Wrapper classes like `Integer`), methods (`add`, `remove`, `set`, `get`, `size`)
- 2D Arrays: Row-major order, nested loop traversals, accessing elements (`arr[row][col]`)
- **Questions**: Writing algorithms to traverse 2D arrays, managing `ArrayList` removals within a loop, comparing array and ArrayList syntax

## Module 5: CS A - Algorithms and Recursion (20%)
- Searching: Linear Search and Binary Search (implementations and preconditions)
- Sorting: Selection Sort and Insertion Sort (tracing the passes)
- Recursion: Base cases, recursive calls, tracing recursive execution, recursive binary search/sorting (Merge Sort conceptual understanding)
- **Questions**: Tracing recursive method outputs, comparing the efficiency of sorting algorithms, implementing a binary search

## Stretch Zone
- Interfaces and Abstract Classes in Java
- Big O Notation (Time complexity analysis of algorithms)
