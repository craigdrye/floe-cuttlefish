# Game Design Basics
**ID**: `gameDesignBasics` · **Discipline**: Computing/Arts · **Year**: 10 (ages 14–15)

## Course Philosophy
An introduction to the theory and practice of game design. This course focuses less on hardcore programming and more on systems thinking, player psychology, and the iterative design process (MDA framework).

## Module 1: The MDA Framework (20%)
- Mechanics: The rules and systems of the game (e.g., jumping, shooting, trading)
- Dynamics: The run-time behavior of the mechanics acting on player inputs (e.g., camping, rushing)
- Aesthetics: The emotional responses evoked in the player (e.g., challenge, fellowship, discovery)
- **Questions**: Analyzing a popular game using the MDA framework, distinguishing mechanics from dynamics

## Module 2: Core Loops and Player Agency (20%)
- The Core Gameplay Loop (e.g., Explore → Fight → Loot → Upgrade)
- Meaningful choices vs illusion of choice (Agency)
- Risk/Reward mechanics
- Feedback loops (Positive/snowballing vs Negative/rubber-banding)
- **Questions**: Identifying core loops, evaluating risk/reward scenarios, designing a negative feedback loop

## Module 3: Level Design & Pacing (20%)
- Teaching the player without tutorials (affordances, signposting, lighting)
- The flow channel (balancing challenge and skill to avoid anxiety or boredom)
- Pacing: peaks and valleys of intensity
- **Questions**: Identifying signposting in level design, applying flow theory to difficulty curves

## Module 4: Economy and Balance (20%)
- Game economies: sources (faucets) and sinks (drains)
- Balancing asymmetric gameplay (e.g., different character classes)
- Power creep and dominant strategies (solving the game)
- **Questions**: Identifying faucets and sinks, analyzing how a dominant strategy ruins balance

## Module 5: Narrative in Games (20%)
- Embedded narrative (cutscenes, dialogue) vs Emergent narrative (stories created by player actions)
- Environmental storytelling (telling a story through the placement of objects in the world)
- Ludonarrative dissonance (when the gameplay contradicts the story)
- **Questions**: Differentiating embedded vs emergent narrative, identifying ludonarrative dissonance

## Stretch Zone
- Introduction to basic logic programming (e.g., state machines, behavior trees)
- The ethics of game design: dark patterns, loot boxes, and addiction mechanics
