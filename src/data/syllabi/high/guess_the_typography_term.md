# Guess the Typography Term

## Course Shape
This design-literacy course identifies typography terms by letter anatomy, spacing, line rhythm, and glyph behavior.

## Chapters
- Type categories: serif and sans serif
- Spacing: kerning, tracking, and leading
- Letter anatomy: baseline and x-height
- Special glyphs: ligatures

## Key Skills
- Use precise vocabulary for text design
- Separate pair spacing, overall spacing, and line spacing
- Notice how small type choices change readability and tone
