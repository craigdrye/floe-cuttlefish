# Guess the Cloud Type

## Course Shape
This Earth-science course identifies clouds through height, shape, layering, texture, rain clues, and storm structure.

## Chapters
- Low clouds: cumulus and stratus
- High clouds: cirrus and cirrostratus
- Middle clouds: altocumulus
- Rain and storm clouds: nimbostratus and cumulonimbus
- Special forms: lenticular clouds

## Key Skills
- Use height, shape, and weather context as clues
- Distinguish puffy, layered, wispy, and towering clouds
- Connect cloud type to likely weather
