# Guess the Machine Shop Tool

## Course Shape
This shop-literacy course identifies machine-shop tools by measuring, turning, milling, drilling, clamping, threading, and finishing functions.

## Chapters
- Measuring tools: calipers and micrometers
- Machines: lathes and drill presses
- Cutting tools: end mills, taps, and reamers
- Work holding: bench vises

## Key Skills
- Connect tool shape to machining task
- Distinguish measuring, cutting, holding, and finishing tools
- Learn practical manufacturing vocabulary without assuming shop access
