# Guess the Architecture Style

## Course Shape
This course teaches building styles through visual grammar: rooflines, columns, arches, ornament, materials, and massing.

## Chapters
- Classical and medieval revivals
- Art Deco and early modern styles
- Bauhaus, Brutalism, and modernism
- Domestic styles: Tudor Revival and Mid-century Modern
- Decorative styles: Art Nouveau

## Key Skills
- Identify style from architectural features
- Connect materials and ornament to historical context
- Describe buildings using precise visual vocabulary
