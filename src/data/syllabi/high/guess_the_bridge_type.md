# Guess the Bridge Type

## Course Shape
This infrastructure-literacy course identifies bridge types by how they carry loads, span distance, use supports, and move when needed.

## Chapters
- Compression forms: arch bridges
- Cable forms: suspension and cable-stayed bridges
- Framed and simple spans: truss and beam bridges
- Special cases: cantilever, bascule, and pontoon bridges

## Key Skills
- Match visible bridge shape to structural behavior
- Distinguish similar cable-supported bridges
- Build everyday engineering vocabulary from real infrastructure
