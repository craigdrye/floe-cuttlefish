# Intro to CS - Python
**Covers**: `col-intro-to-cs---python`, `col-computer-programming`, `col-computers-and-the-internet`
**Discipline**: Computing · **Year**: 10 (ages 14–15)

## Course Philosophy
This course introduces students to the fundamentals of computer science and programming using Python. It covers algorithmic thinking, basic data structures, control flow, and an introduction to how the internet works.

## Module 1: The Internet & How Computers Work (15%)
- Bits and Bytes: How computers represent data (binary, ASCII, Unicode)
- Hardware vs Software: CPU, RAM, Storage, input/output devices
- How the Internet works: IP addresses, packets, routers, DNS, and TCP/IP
- Cybersecurity basics: Phishing, malware, encryption (public/private key concepts)
- **Questions**: Binary to decimal conversion, explaining DNS, identifying hardware components

## Module 2: Python Basics & Variables (20%)
- Print statements and string formatting
- Variables, naming conventions, and data types (Integers, Floats, Strings, Booleans)
- Basic operators (Arithmetic: +, -, *, /, //, %, **; Assignment: =, +=, -=)
- Taking user input and type casting (e.g., `int(input())`)
- Commenting and code readability
- **Questions**: Evaluating arithmetic expressions in Python, predicting output of variable reassignment, fixing type errors

## Module 3: Control Flow (Conditionals & Loops) (25%)
- Boolean logic: `and`, `or`, `not`, and comparison operators (`==`, `!=`, `<`, `>`, `<=`, `>=`)
- `if`, `elif`, and `else` statements
- `while` loops: infinite loops, loop conditions, `break`, and `continue`
- `for` loops: iterating over ranges (`range(start, stop, step)`) and strings
- Nested loops and nested conditionals
- **Questions**: Tracing loop execution, evaluating complex boolean expressions, predicting `if/else` outcomes

## Module 4: Data Structures (Lists & Strings) (20%)
- String manipulation: indexing, slicing (`string[start:stop:step]`), and common methods (`.lower()`, `.upper()`, `.replace()`)
- Lists: creating lists, zero-based indexing, slicing
- List methods: `.append()`, `.remove()`, `.pop()`, `.insert()`, `len()`
- Iterating through lists using `for` loops
- Mutability (lists) vs Immutability (strings)
- **Questions**: Predicting output of list slicing, tracing list modifications, iterating through arrays

## Module 5: Functions & Algorithmic Thinking (20%)
- Defining functions using `def`
- Parameters vs Arguments
- Return statements vs Print statements
- Variable scope: local vs global variables
- Algorithmic thinking: breaking down a problem into smaller steps
- Introduction to simple algorithms: linear search, finding the max/min in a list
- **Questions**: Predicting output of function calls, identifying scope errors, writing simple algorithmic logic

## Stretch Zone
- Introduction to Dictionaries (Key-Value pairs)
- Basic file I/O (opening, reading, and writing to text files)
- Introduction to Object-Oriented Programming (Classes and Objects)
