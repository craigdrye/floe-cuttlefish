# Guess the Body Part

## Course Shape
This is a visual anatomy identification course for medicine-curious learners. Each question uses a cute cartoon clue and a more literal field-guide-style clue, then asks for the body part, organ, bone, vessel, nerve, or anatomical landmark.

## Chapters
- Organs: heart, lungs, brain, kidney, liver, and stomach
- Bones: femur, skull, and rib cage
- Vessels and nerves: aorta and spinal cord
- Landmarks: diaphragm

## Key Skills
- Match structure to function without overclaiming clinically
- Use location, shape, tissue type, and system clues
- Build visual anatomy vocabulary for biology, medicine, nursing, and allied health foundations

## Safety Note
This course teaches anatomy recognition and vocabulary. It is not a diagnostic tool and should not be used for medical decisions.
