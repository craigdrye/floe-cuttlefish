# Guess the Sewing Tool

## Course Shape
This making-skills course identifies sewing tools by cutting, measuring, marking, stitch removal, threading, hand protection, and machine function.

## Chapters
- Cutting and opening seams: fabric shears, rotary cutters, and seam rippers
- Measuring and marking: tape measures and tailor’s chalk
- Hand sewing helpers: thimbles and needle threaders
- Machine basics: bobbins

## Key Skills
- Match sewing tools to the job they make easier
- Distinguish cutting, marking, measuring, and threading tools
- Build craft vocabulary for repair and making
