# Guess the Hand Tool

## Course Shape
This practical course teaches common hand tools from shape, working action, and job.

## Chapters
- Gripping tools: wrenches and pliers
- Cutting tools: saws and chisels
- Measuring tools: calipers and levels
- Driving tools: hex keys and star drivers
- Tool safety and choosing the right tool for the task

## Key Skills
- Match tool shape to function
- Use the right vocabulary for repair and making
- Distinguish similar tools by what they do
