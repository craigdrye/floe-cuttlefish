# Guess the Surgical Tool

## Course Shape
This medical-vocabulary course identifies surgical tools by task: cutting, grasping, clamping, retracting, closing, examining, accessing, or suctioning.

## Chapters
- Cutting and grasping: scalpel and forceps
- Control and exposure: hemostat and retractor
- Closure and examination: suture needle and speculum
- Access and suction: trocar and Yankauer suction tip

## Key Skills
- Match instrument shape to clinical function
- Distinguish similar-looking tools by use case
- Learn operating-room vocabulary without pretending to perform procedures
