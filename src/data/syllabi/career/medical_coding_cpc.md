# Medical Coding CPC
**ID**: `medicalCodingCpc` · **Discipline**: Healthcare · **Level**: Career

## Course Aim
Prepare learners for CPC-style medical coding and entry-level professional practice: reading documentation, assigning ICD-10-CM, CPT, and HCPCS Level II codes, applying modifiers, and avoiding preventable claim and compliance errors.

## Course Design Notes
AAPC CPC materials emphasize anatomy, medical terminology, ICD-10-CM, CPT, HCPCS Level II, modifiers, guidelines, and specialty coding. This syllabus keeps those exam pillars while adding claim-cleanliness, query judgment, audit rationale, and production habits.

## Chapter 1: Coding Profession, Compliance, and Reference Books
- **Key concepts**: coder role, official guidelines, code books, payer rules, compliance, documentation integrity.
- **Practice questions**: What code set applies? Which guideline controls? What should a coder never infer?
- **Work product**: a coding-reference map for ICD-10-CM, CPT, HCPCS Level II, modifiers, and guidelines.

## Chapter 2: Medical Terminology and Anatomy
- **Key concepts**: body systems, root words, prefixes, suffixes, laterality, anatomy landmarks, procedure language.
- **Practice questions**: What body system is involved? Which term changes the procedure or diagnosis? What detail is missing?
- **Work product**: a terminology-to-code clue sheet organized by body system.

## Chapter 3: ICD-10-CM Diagnosis Coding
- **Key concepts**: main term lookup, tabular verification, specificity, laterality, sequencing, combination codes, excludes notes.
- **Practice questions**: What is the principal diagnosis? Is the code specific enough? Which note blocks or redirects the choice?
- **Work product**: ten diagnosis coding cases with rationale and guideline citations.

## Chapter 4: CPT Procedure Coding
- **Key concepts**: E/M basics, surgery sections, anesthesia, radiology, pathology/lab, medicine, add-on codes.
- **Practice questions**: What service was performed? Which approach or extent matters? Is an add-on code allowed?
- **Work product**: a CPT coding worksheet with documentation clue, code path, and final code.

## Chapter 5: HCPCS Level II and Supplies
- **Key concepts**: supplies, drugs, durable medical equipment, ambulance, temporary codes, units, payer nuance.
- **Practice questions**: What item or service is outside CPT? How many units apply? What documentation supports medical necessity?
- **Work product**: an HCPCS coding mini-set with units and documentation rationale.

## Chapter 6: Modifiers, Bundling, and NCCI Logic
- **Key concepts**: modifier purpose, distinct service, laterality, professional/technical components, bundling, NCCI edits.
- **Practice questions**: Does a modifier explain a real distinction? Is this bundled? Would the claim invite denial?
- **Work product**: a modifier decision table for common outpatient scenarios.

## Chapter 7: Specialty Coding and Exam Timing
- **Key concepts**: cardiology, orthopedics, obstetrics, gastroenterology, integumentary, respiratory, time management.
- **Practice questions**: Which specialty rules appear? What can be solved quickly? Which case should be skipped and returned to?
- **Work product**: a timed practice log by section, accuracy, and error type.

## Chapter 8: Claims, Denials, Queries, and Audits
- **Key concepts**: clean claims, denials, medical necessity, provider queries, audit trail, overcoding, undercoding.
- **Practice questions**: Is a query needed? What rationale supports the code? What would an auditor ask?
- **Work product**: an audit defense note explaining code selection, missing documentation, and compliant next step.

## Capstone
Code a mixed outpatient case packet. Submit diagnosis, procedure, HCPCS, modifier, and rationale selections; flag documentation gaps; write one compliant provider query; and complete a denial-prevention checklist.

## Assessment Ideas
- Diagnosis and procedure drills graded on guideline citation and specificity.
- Modifier cases graded on compliance and bundling logic.
- Capstone graded on accuracy, speed, documentation judgment, and audit defensibility.
