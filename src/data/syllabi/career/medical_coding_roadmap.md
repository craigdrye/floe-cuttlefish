# Medical Coding Roadmap
**ID**: `medicalCodingRoadmap` - **Discipline**: Healthcare - **Level**: Career

## Course Aim
Prepare learners for entry-level medical coding work: reading documentation, choosing diagnosis and procedure codes, applying guidelines and modifiers, spotting documentation gaps, and explaining coding rationale clearly.

## Course Design Notes
Route questions here when they test ICD-10-CM, CPT, HCPCS, modifiers, guideline lookup, documentation support, compliant queries, audits, or denial-prevention logic. Keep the emphasis on evidence in the record, not guessing from clinical vibes.

## Chapter 1: Coding Role and Reference Habits
- **Key concepts**: coder role, code sets, official guidelines, payer rules, compliance, audit trail, documentation integrity.
- **Practice questions**: Which reference applies? What can be coded from the record? What must never be inferred?
- **Work output**: coding reference map with when to use each book, guideline, and payer source.

## Chapter 2: Diagnosis Coding
- **Key concepts**: ICD-10-CM, main term, tabular verification, specificity, laterality, sequencing, excludes notes.
- **Practice questions**: What condition is documented? Is the code specific enough? Which note changes the choice?
- **Work output**: diagnosis coding worksheet with lookup path, final code, and rationale.

## Chapter 3: Procedure and Supply Coding
- **Key concepts**: CPT, HCPCS Level II, E/M basics, procedure description, units, supplies, medical necessity.
- **Practice questions**: What service was performed? What detail changes the code? What supports the billed units?
- **Work output**: procedure coding worksheet with documentation clue, code path, and final selection.

## Chapter 4: Modifiers, Bundling, and Claim Edits
- **Key concepts**: modifier, NCCI edit, bundled service, distinct procedural service, laterality, professional component.
- **Practice questions**: Does the modifier tell the truth? Is the edit correct? What documentation supports separate payment?
- **Work output**: modifier decision table for common outpatient scenarios.

## Chapter 5: Queries, Audits, and Denial Prevention
- **Key concepts**: provider query, ambiguous documentation, overcoding, undercoding, audit defense, denial trend.
- **Practice questions**: Is a query needed? What rationale would satisfy an auditor? Which mistake could cause a denial?
- **Work output**: audit defense note with codes, evidence, gaps, and compliant query if needed.

## Capstone
Code a mixed encounter packet. Assign diagnosis, procedure, HCPCS, and modifier codes; flag documentation gaps; write one compliant query; and produce a denial-prevention checklist.
