# Customs Broker License
**ID**: `customsBroker` · **Discipline**: Operations

## Course Philosophy
This syllabus prepares learners for customs-broker style reasoning: classification accuracy, origin logic, entry diligence, and compliance records that survive scrutiny.

## Module 1: Classification and Valuation
- HTS/HS code structure and classification method
- Customs value, invoices, assists, and dutiable elements
- Common classification and valuation mistakes

## Module 2: Origin and Entry Process
- Country-of-origin rules and documentation logic
- Entry summaries, timelines, and broker responsibilities
- Special programs, exceptions, and trade-agreement basics

## Module 3: Compliance Operations
- Recordkeeping, audits, corrections, and broker ethics
- Communication with importers, carriers, and agencies
- Escalation when documentation does not reconcile

## Stretch Zone
- AD/CVD issues, drawback, FTZs, and post-entry amendments
