# UX Design
**ID**: `uxDesign` · **Discipline**: Design

## Course Philosophy
This syllabus develops user-centered design judgment, from flow mapping and wireframing through interaction reasoning and portfolio storytelling.

## Module 1: Problem Framing and User Flows
- User goals, constraints, and task analysis
- Flows, information architecture, and friction mapping
- Designing for clarity, affordance, and reduced cognitive load

## Module 2: Interface Structure
- Wireframes, component hierarchy, and state design
- Interaction patterns, feedback, and error prevention
- Accessibility and responsive behavior as design fundamentals

## Module 3: Portfolio and Review
- Case studies: problem, process, decisions, and outcome
- Explaining tradeoffs to product and engineering partners
- Critique habits and iterative improvement

## Stretch Zone
- Service design, experimentation, and systems-level UX consistency
