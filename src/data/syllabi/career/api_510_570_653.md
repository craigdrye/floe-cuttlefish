# API 510/570/653 Inspector
**ID**: `apiInspectors` - **Discipline**: Energy - **Level**: Career

## Course Aim
Prepare learners for API 510, API 570, and API 653 inspection work: code navigation, corrosion and damage mechanisms, inspection planning, repairs and alterations, pressure tests, records, and fitness-for-service reasoning.

## Course Design Notes
API inspector certifications emphasize code structure, pressure equipment integrity, inspection intervals, repair requirements, testing, documentation, and damage-mechanism awareness. This syllabus integrates API 510 pressure vessels, API 570 piping, and API 653 tanks so learners practice finding the controlling rule and turning inspection evidence into defensible recommendations.

## Chapter 1: Inspector Role, Code Scope, and Reference Navigation
- **Key concepts**: authorized inspector, owner-user program, API 510, API 570, API 653, ASME references, jurisdiction, code hierarchy.
- **Practice questions**: Which code applies? What equipment is in scope? Which referenced document controls the next step?
- **Work product**: a code-navigation map for vessels, piping, tanks, repairs, testing, and records.

## Chapter 2: Equipment Design Basics and Inspection Data
- **Key concepts**: pressure vessels, process piping, aboveground storage tanks, design pressure, MAWP, minimum thickness, service conditions.
- **Practice questions**: What equipment data is needed? Which design variable controls the calculation? What history changes the inspection plan?
- **Work product**: an equipment data sheet with design basis, service, materials, thickness readings, and open questions.

## Chapter 3: Corrosion and Damage Mechanisms
- **Key concepts**: general corrosion, localized corrosion, pitting, erosion, cracking, CUI, fatigue, brittle fracture, API 571 awareness.
- **Practice questions**: Which damage mechanism fits the evidence? Where should inspection focus next? What operating condition is the driver?
- **Work product**: a damage-mechanism worksheet linking symptoms, likely causes, inspection methods, and mitigation.

## Chapter 4: Inspection Planning, Intervals, and Methods
- **Key concepts**: internal inspection, external inspection, on-stream inspection, thickness monitoring, RBI, NDE, corrosion rates, remaining life.
- **Practice questions**: When is the next inspection due? Which NDE method is appropriate? Is the interval justified by data?
- **Work product**: an inspection plan with interval basis, inspection locations, methods, access needs, and acceptance criteria.

## Chapter 5: API 510 Pressure Vessel Inspection
- **Key concepts**: MAWP, corrosion allowance, pressure vessel repairs, alterations, rerating, pressure relief devices, pressure test decisions.
- **Practice questions**: Is the vessel fit for continued service? Does the change count as an alteration? Is a pressure test required?
- **Work product**: a vessel inspection report with thickness evaluation, repair recommendation, and test rationale.

## Chapter 6: API 570 Piping Inspection
- **Key concepts**: piping circuits, injection points, deadlegs, CMLs, class of service, temporary repairs, pressure testing, leak testing.
- **Practice questions**: Which circuit has the highest risk? Where should CMLs be added? Is the proposed repair acceptable?
- **Work product**: a piping circuit assessment with CML map, corrosion-rate calculation, repair plan, and records update.

## Chapter 7: API 653 Tank Inspection
- **Key concepts**: tank shell, bottom, roof, settlement, brittle fracture, out-of-service inspection, in-service inspection, repairs.
- **Practice questions**: Does the tank require out-of-service inspection? What settlement condition matters? Which repair standard applies?
- **Work product**: a tank inspection summary with findings, settlement review, repair scope, and next inspection date.

## Chapter 8: Fitness-for-Service, Records, and Exam Readiness
- **Key concepts**: API 579 awareness, flaw evaluation, rerating, inspection records, repair documentation, pressure-test records, exam timing.
- **Practice questions**: Is a fitness-for-service assessment needed? What record proves compliance? Which code section answers fastest?
- **Work product**: a fitness-for-service decision memo and a practice-exam error log sorted by code, topic, and lookup error.

## Capstone
Prepare an integrity packet for a fictional facility with one vessel, one piping circuit, and one storage tank. Include code-scope decisions, inspection plans, damage-mechanism notes, thickness calculations, repair recommendations, pressure-test rationale, records index, and fitness-for-service escalation decisions.

## Assessment Ideas
- Code-navigation drills graded on speed, correct reference, and section-level support.
- Thickness, interval, and remaining-life calculations graded on setup, units, assumptions, and conclusion.
- Capstone graded on inspection judgment, repair defensibility, record quality, and fitness-for-service escalation.
